# Branch Info

Αυτό είναι το **δικό μου branch** για development.

Κάνω ότι θέλω εδώ και μετά από **συνεννόηση** θα merge στο main.

Κάθε άτομο έχει το **δικό του branch**.

---

## Βασικές Git Εντολές

```bash
# Δες σε ποιο branch είσαι
git branch

# Δημιούργησε νέο branch
git checkout -b my-branch

# Άλλαξε branch
git checkout branch-name

# Δες τι άλλαξες
git status

# Πρόσθεσε αλλαγές
git add .

# Commit
git commit -m "μήνυμα"

# Στείλε στο remote
git push
```
