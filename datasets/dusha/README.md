# Dusha Dataset

## 📊 Χαρακτηριστικά

- **Μέγεθος**: 58 GB
- **Samples**: ~300,000 audio recordings
- **Διάρκεια**: ~350 hours
- **Γλώσσα**: Russian
- **Έτος**: 2022

## Συναισθήματα

- **Angry** (θυμωμένος)
- **Happy** (χαρούμενος)
- **Sad** (λυπημένος)
- **Neutral** (ουδέτερος)

## Dimensions

- **Arousal** (διέγερση)
- **Valence** (συναίσθημα)
