# TESS Dataset
Toronto Emotional Speech Set (TESS)

## Χαρακτηριστικά

- **Μέγεθος**: ~281 MB
- **Δείγματα**: 2800 data points (audio files)
- **Ηθοποιοί** : 2 (γυναίκες)
- **Ηλικιακό εύρος**: 26 και 64
- **Γλώσσα**: English
- **Προτάσεις**: a set of 200 target words were spoken in the carrier phrase "Say the word _"

## Συναισθήματα (Emotion Labels)

- **Anger** (θυμός)
- **Disgust** (αηδία)
- **Fear** (φόβος)
- **Happy** (χαρά)
- **Neutral** (ουδέτερο)
- **Sad** (λύπη)
- **Pleasant Surprise** (ευχάριστη έκπληξη)
